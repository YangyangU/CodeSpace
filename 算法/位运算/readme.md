# 十进制转二进制


# 所有的位运算方法都是为二进制服务的

- 按位与（&）
    将两个数字的二进制的每一位进行 & 操作，如果两个位都为1,则该位的结果为1,否则为0.

- 按位或（|）
    将两个数字的二进制的每一位进行 | 操作，如果两个位至少有一个为1,则该位的结果为1,否则为0.

- 按位异或（^）
    将两个数字的二进制的每一位进行 ^ 操作，如果两个位的二进制值不同,则该位的结果为1,否则为0.

- 按位非（~）
    将一个数字的二进制每一位进行取反操作,即0变成1,1变成0，将得到的值转成32位有符号整型
    - 任何小数前面加两个~~得到向下取整数
        ~~2.5 == 2
