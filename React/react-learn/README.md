# React 了解

- Vue 和 React 有什么相同点和不同点
  - 都是MVVM的现代前端框架，React早于Vue
  - VDOM，diff 组件化

  - Vue上手简单，主要得益于良好的API
  - React JSWAY
    - 组件就是函数，返回HTML就是组件
    - JSX
  - Vue 支持指令 v-model 双向数据绑定
    react 觉得双向数据绑定性能不好， 不支持

- 为什么要用TS
  - JS是弱类型语言，ts添加了类型系统
  - 减少很多错误出现
  - 类型推导
  - 静态编译
    出了错还不知道，反复调试
  - ts更适合大型多人协作项目开发
